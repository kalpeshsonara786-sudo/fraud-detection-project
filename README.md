# 🔍 Credit Card Fraud Detection System

## 📌 Project Overview
This project analyzes 10,000 credit card transactions 
to detect fraudulent activity using Machine Learning.

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib & Seaborn
- Plotly
- Scikit-learn (ML)

## 📊 Key Findings
- Fraud Rate: 1.51% (151 transactions)
- Peak Fraud Hour: 12 AM - 3 AM
- Riskiest Category: Grocery
- Best Fraud Indicator: Device Trust Score

## 🤖 ML Model Results
- Algorithm: Random Forest
- Accuracy: 99.8%

## 📈 Dashboards
- Fraud Analysis Dashboard
- Hidden Fraud Detection
- Bank Loan Prediction (100% Accuracy)
